Module Hierarchy

  • 3DPhotovoltaics: Root directory.
    • 3DPhotovoltaics.Data_Output: Holds all classes and methods relevant to data output
      • 3DPhotovoltaics.Data_Output.Analysis
      • 3DPhotovoltaics.Data_Output.Raw_Output_Test: Contains some methods for testing data output functionality
      • 3DPhotovoltaics.Data_Output.Stat: Contains the stats class
      • 3DPhotovoltaics.Data_Output.Statistics
    • 3DPhotovoltaics.Main: Contains the main method which does some basic setup and conclusion steps
    • 3DPhotovoltaics.Misc: Contains some miscellaneous files that were worth keeping around.
      • 3DPhotovoltaics.Misc.XMLInputParser
      • 3DPhotovoltaics.Misc.walker_parser
    • 3DPhotovoltaics.Orbit: Contains classes and methods relevant to the simulations orbit
      • 3DPhotovoltaics.Orbit.Orbit: Contains the abstract orbit class
      • 3DPhotovoltaics.Orbit.OrbitWithSunshine: Contains class for an orbit that models the ISS orbiting the earth
      • 3DPhotovoltaics.Orbit.SimpleOrbit: Contains class for a simple orbit.
      • 3DPhotovoltaics.Orbit.Transform: Contains utility class for matrix transformations
    • 3DPhotovoltaics.Simulation: Holds classes and functions that are directly related to the principle event handling of the simulation
      • 3DPhotovoltaics.Simulation.Photon: Holds the Photon class
      • 3DPhotovoltaics.Simulation.Record: Holds the Record class
      • 3DPhotovoltaics.Simulation.Simulation: Contains the run method for running the simulation
    • 3DPhotovoltaics.Tests: Contains modules that are related to testing
      • 3DPhotovoltaics.Tests.BasicTests: Script that runs some very basic tests.
    • 3DPhotovoltaics.Tower: Holds all classes and methods directly related to Tower
      • 3DPhotovoltaics.Tower.Cylinder: Holds the Cylinder class
      • 3DPhotovoltaics.Tower.Floor: Holds the floor class
      • 3DPhotovoltaics.Tower.Material: Holds the Material class
      • 3DPhotovoltaics.Tower.Plane: Holds the Plane class
      • 3DPhotovoltaics.Tower.Tower: Holds the Tower class
      • 3DPhotovoltaics.Tower.Wall: Holds the Wall class
    • 3DPhotovoltaics.XML_Input: Holds all of the classes and methods relevant in parsing the users XML input
      • 3DPhotovoltaics.XML_Input.GraphSettings: Holds the GraphSettings class
      • 3DPhotovoltaics.XML_Input.OutputSettings: Holds the OutputSettings class
      • 3DPhotovoltaics.XML_Input.SimulationSettings: Holds the SimulationSettings class
      • 3DPhotovoltaics.XML_Input.XML_Reader: Contains methods for parsing the users XML file and validating it
    • 3DPhotovoltaics.make_flicker_graphs: Simple script that was used to prdocue some graphs to compare with those shown by Jack Flicker in his derivation of power gain paper
    • 3DPhotovoltaics.sgp4: Track earth satellite TLE orbits using up-to-date 2010 version of SGP4
      • 3DPhotovoltaics.sgp4.earth_gravity: Three earth-gravity models for use with SGP4.
      • 3DPhotovoltaics.sgp4.ext: Utility routines from "sgp4ext.cpp".
      • 3DPhotovoltaics.sgp4.io: Read the TLE earth satellite file format.
      • 3DPhotovoltaics.sgp4.model: The Satellite class.
      • 3DPhotovoltaics.sgp4.propagation: The sgp4 procedures for analytical propagation of a satellite.
      • 3DPhotovoltaics.sgp4.setup
      • 3DPhotovoltaics.sgp4.tests: Test suite for SGP4.
